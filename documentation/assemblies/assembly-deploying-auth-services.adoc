= Deploying authentication services

{ProductName} requires at least one authentication service to be deployed. 
ifndef::AuthServices[]
The authentication service can be `standard` ({AuthProductName}) or `external` (not managed by {ProductName}).
endif::AuthServices[]
ifdef::AuthServices[]
The authentication service can be `none` (allow all), `standard` ({AuthProductName}), or `external` (not managed by {ProductName}).
endif::AuthServices[]
